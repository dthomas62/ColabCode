2. The JSONL Generator Script
This script reads your CSV and creates the requests.jsonl file. It includes the Google Search tool, which is mandatory if you want accurate social media stats in 2026.
import pandas as pd
import json

# Load your data
df = pd.read_csv("brands.csv")

# Configuration (The "Same Parameters" for every request)
MODEL_CONFIG = {
    "temperature": 1.0,
    "thinking_config": {"thinking_level": "MEDIUM"} 
}

jsonl_output = "brand_research_batch.jsonl"

with open(jsonl_output, "w") as f:
    for index, row in df.iterrows():
        # Create a dynamic prompt using your spreadsheet columns
        prompt_text = (
            f"Perform a market analysis for the brand '{row['brand']}' "
            f"within the '{row['category']}' category for the '{row['market']}' market. "
            f"Provide current social media engagement stats and a summary of "
            f"the brand's perception in this specific region."
        )

        # Build the JSON object for this specific line
        request_line = {
            "key": f"req_{index}_{row['brand'].replace(' ', '_')}", # Unique ID for results
            "request": {
                "contents": [{"parts": [{"text": prompt_text}], "role": "user"}],
                "tools": [{"google_search": {}}], # Enable 2026 Search Grounding
                "generation_config": MODEL_CONFIG
            }
        }
        
        # Write as a single line in the JSONL file
        f.write(json.dumps(request_line) + "\n")

print(f"Done! Created {len(df)} requests in {jsonl_output}")

• Prompt Injecting: By using f-strings (f"..."), the script automatically creates 100 unique prompts.
• Google Search Tool: Since Gemini 3.1 Pro’s internal knowledge cutoff is January 2025, the search tool is what allows the model to find "live" 2026 social media stats.
